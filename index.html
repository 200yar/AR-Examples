AR.js Examples
The goal of this collection is to provide a set of basic and instructive examples that introduce the various features in AR.js.

Viewing these examples will require a device with a camera and one or both of the Hiro and Kanji marker patterns (either printed or displayed on a second device).

Source code hosted at GitHub. •

Hello Cube

A basic scene that superimposes a cube on a Hiro marker.


Image

An image is positioned to cover a Hiro marker.


Rotating Globe

A rotating globe (illuminated by a point light) is positioned above a Hiro marker.
